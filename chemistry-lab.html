👋 这是一个"化学虚拟实验室"，不是动画片

这里复刻真实的化学实验：你要自己选装置、控制变量、观察颜色与气泡、读天平和量筒、把现象和数据记下来、再用证据下结论——和中考实验探究题的考法完全一致。先动手做，再做题，印象最深。

选一个实验进实验室

化学实验探究的"五步法"（每个实验都按这个走）

  1. 提出问题 / 明确目的：要制什么、验证什么、探究哪个因素？
  2. 设计实验：选发生/收集装置，用控制变量法，想好对照组。
  3. 动手操作：加药品、加热、滴加、过滤……注意操作顺序和安全。
  4. 观察记录：颜色、气泡、沉淀、温度、读数都要如实记录。
  5. 分析结论：用现象/数据作证据下结论，并做误差与评价分析。

背景说明

面向 2031 合肥/安徽中考，人教版九年级化学。这里挑的都是中考实验探究高频考点：气体制取、燃烧条件、质量守恒、电解水、酸碱中和、金属活动性、溶液配制、粗盐提纯。模拟器里的数值和现象按真实化学规律计算，可放心拿来理解原理、对照真题。

🅞 氧气的实验室制取与收集

初中第一个"气体制取"实验。要学会根据反应物状态和条件选发生装置、根据气体性质选收集方法，并掌握验满、检验、注意事项——这些都是中考实验题的固定考点。

三个常见反应（写对化学方程式）

  • 加热高锰酸钾：2KMnO₄ →(Δ) K₂MnO₄ + MnO₂ + O₂↑（固+固，加热型）
  • 双氧水+二氧化锰：2H₂O₂ →(MnO₂) 2H₂O + O₂↑（固+液，常温型，MnO₂是催化剂）
  • 加热氯酸钾：2KClO₃ →(MnO₂,Δ) 2KCl + 3O₂↑（固+固，加热型）

装置选择口诀

  • 发生装置看状态和条件：固固加热 → 试管+酒精灯；固液常温 → 锥形瓶+分液漏斗。
  • 收集装置看溶解性和密度：O₂不易溶于水 → 排水法(更纯)；O₂密度比空气大 → 向上排空气法。
  • 检验O₂：带火星的木条伸入，复燃。
  • 验满：排空气法把带火星木条放瓶口复燃即满；排水法看气泡从瓶口往外冒。

💨 二氧化碳的实验室制取与性质

药品选择是重点考点：为什么用大理石(石灰石)和稀盐酸，不用稀硫酸、不用浓盐酸、不用碳酸钠？再加上检验、验满和"倾倒灭蜡烛"这个经典性质实验。

反应原理与药品选择

  • 原理：CaCO₃ + 2HCl = CaCl₂ + H₂O + CO₂↑（固+液，常温）。
  • ❌ 不用稀硫酸：生成微溶的 CaSO₄ 覆盖在石灰石表面，阻止反应继续。
  • ❌ 不用浓盐酸：易挥发，使收集的 CO₂ 混有 HCl 气体不纯。
  • ❌ 不用碳酸钠粉末：反应太快，不便控制。

收集 · 检验 · 验满 · 性质

  • 只能向上排空气法：CO₂能溶于水且与水反应，密度比空气大。不能排水法。
  • 检验：通入澄清石灰水变浑浊 CO₂+Ca(OH)₂=CaCO₃↓+H₂O。
  • 验满：把燃着的木条放瓶口，熄灭即满。
  • 性质：不燃烧、不支持燃烧、密度比空气大 → 倾倒时下层蜡烛先灭。与水反应 CO₂+H₂O=H₂CO₃ 使紫色石蕊变红。

🔥 探究燃烧的条件

燃烧需要三个条件同时满足。这个经典的"白磷红磷"实验用控制变量法，一次只改变一个因素，分别证明缺哪个条件就烧不起来。

燃烧的三个条件（缺一不可）

  • ① 有可燃物；
  • ② 可燃物与氧气(空气)接触；
  • ③ 温度达到可燃物的着火点。
  • 白磷着火点约 40℃，红磷约 240℃。热水(约80℃)能点着白磷，点不着红磷。

三处对照说明什么

  • 铜片白磷 vs 铜片红磷：都接触O₂、温度相同 → 只差着火点。白磷燃、红磷不燃 → 证明要达到着火点。
  • 铜片白磷 vs 水中白磷：都是白磷、温度都够 → 只差是否接触O₂。水中白磷不燃 → 证明要接触氧气。
  • 给水中白磷通O₂ → 立刻"水下燃烧" → 反证接触O₂的重要性。

⚖️ 探究质量守恒定律

参加化学反应的各物质质量总和，等于反应后生成各物质质量总和。难点是：为什么有的反应在敞口天平上"看起来变轻/变重"？关键看有没有气体进出。

定律 & 微观本质

  • 质量守恒：m(反应物) = m(生成物)，一切化学反应都遵守。
  • 微观原因：反应前后原子的种类、数目、质量都不变，只是重新组合。
  • "五个不变"：元素种类、原子种类/数目、各原子质量、总质量；"两个一定变"：分子种类、物质种类。

为什么天平会"变"

  • 有气体生成并逸出（如 Na₂CO₃+盐酸）→ 敞口天平变小。
  • 有气体参加反应（吸收空气）（如镁燃烧）→ 敞口固体变大。
  • 要验证守恒，凡涉及气体的反应必须在密闭装置里称量。守恒定律本身从没被破坏。

💧 电解水 · 探究水的组成

给水通直流电，水会分解成两种气体。通过检验两极气体、比较体积，证明水是由氢、氧两种元素组成的。记住八字口诀："正氧负氢、氢二氧一"。

原理 & 现象

  • 反应：2H₂O →(通电) 2H₂↑ + O₂↑。
  • 负极产生氢气H₂，正极产生氧气O₂。
  • 体积比 V(H₂) : V(O₂) = 2 : 1（负极气体多，是正极的2倍）。
  • 常加少量稀硫酸或NaOH，增强水的导电性，加快分解。

验气 & 结论

  • 正极O₂：带火星木条伸入 → 复燃。
  • 负极H₂：移近火焰 → 淡蓝色火焰(纯度不够会"噗"地爆鸣)。
  • 结论：水是由氢元素和氧元素组成的化合物。
  • 微观：化学变化中分子可分、原子不可分（水分子分成氢、氧原子再重组）。

🧫 酸碱中和反应 · pH与指示剂

向滴有酚酞的NaOH溶液里逐滴加盐酸，看着红色慢慢褪去——恰好变无色时酸碱中和完全。同时用 pH 监测整个过程，理解指示剂变色和 pH 曲线。

中和反应 & 指示剂

  • 中和反应：酸+碱→盐+水，NaOH + HCl = NaCl + H₂O，放热。
  • 紫色石蕊：遇酸变红、遇碱变蓝、中性紫色。
  • 无色酚酞：遇碱变红、中性和酸性都无色。
  • 判断恰好完全：滴入酸使酚酞红色刚好褪为无色。

pH 的意义

  • pH<7 酸性，pH=7 中性，pH>7 碱性。
  • pH 越小酸性越强，pH 越大碱性越强。
  • 测 pH：用玻璃棒蘸取待测液滴在 pH 试纸上，与标准比色卡对照（不能把试纸直接伸进溶液）。
  • 本实验滴入盐酸时 pH 从大变小，在恰好中和处(pH=7)附近会突变。

🔩 探究金属活动性顺序

把不同金属放进稀盐酸、硫酸铜溶液、硝酸银溶液里，比较反应快慢、能否置换，就能排出金属的活动性强弱顺序。这是中考"金属"主题的核心实验。

🥤 配制一定溶质质量分数的溶液

四步走：计算 → 称量 → 量取 → 溶解。重点是会算溶质和水各多少，以及分析哪些操作会让浓度偏大或偏小（误差分析是中考高频）。

计算公式 & 步骤

  • 溶质质量分数 = 溶质质量 ÷ 溶液质量 ×100%。
  • 溶质质量 = 溶液质量 × 质量分数；溶剂(水)质量 = 溶液质量 − 溶质质量。
  • 水的体积 = 水的质量 ÷ 密度(1g/mL)，用量筒量取。
  • 仪器：托盘天平、药匙、量筒、胶头滴管、烧杯、玻璃棒。

误差分析（会偏大/偏小？）

  • 量水时仰视读数 → 实际水偏多 → 浓度偏小。
  • 量水时俯视读数 → 实际水偏少 → 浓度偏大。
  • 砝码与药品放反(且动了游码) → 药品偏少 → 浓度偏小。
  • 药品撒落、烧杯内壁有水 → 浓度偏小。

🧂 粗盐提纯 · 过滤与蒸发

把混有泥沙的粗盐变成精盐：溶解 → 过滤 → 蒸发结晶。过滤的"一贴二低三靠"和蒸发的"何时停止加热"是中考必考操作细节。

三步操作 & 仪器

  • 溶解：粗盐入水，用玻璃棒搅拌加速溶解（泥沙不溶）。
  • 过滤：分离不溶性杂质（泥沙留在滤纸上，食盐随水通过）。
  • 蒸发：蒸发皿加热，玻璃棒搅拌，析出 NaCl 晶体。
  • 仪器：烧杯、玻璃棒、漏斗、滤纸、铁架台、蒸发皿、酒精灯。

关键操作细节

  • 过滤"一贴"：滤纸紧贴漏斗内壁；"二低"：滤纸边缘低于漏斗口、液面低于滤纸边缘；"三靠"：烧杯靠玻璃棒、玻璃棒靠三层滤纸、漏斗下端靠烧杯内壁。
  • 玻璃棒作用：溶解时搅拌、过滤时引流、蒸发时搅拌防飞溅。
  • 蒸发：当出现较多固体时就停止加热，用余热蒸干（防止飞溅）。
  • 滤液仍浑浊原因：滤纸破损/液面高于滤纸边缘/仪器不干净 → 需重新过滤。
