真实实验的"四步法"（每个实验都按这个走）

  1. 明确目的与原理：要测什么？依据哪个公式？（如 R=U/I）
  2. 设计与操作：连电路、选量程、用控制变量法，操作变阻器/光屏等。
  3. 多次测量记录：点"记录数据"把每组读数填进数据表（实验讲究多次测量取平均、减小误差）。
  4. 处理数据下结论：用你测的数据算平均值/画图线，得出规律，并做误差分析。

🔌 电路搭建实验室

真正的开放式电路沙盒：自己拖元件、点接线柱连导线、随便怎么接，背后有真实的电路求解器（节点分析法）实时算出每个元件的电流、电压，并自动判断 通路 / 断路 / 短路。

怎么玩

  • 添加元件：点上方按钮添加电池/灯泡/电阻/开关/变阻器/电流表/电压表。
  • 连线：切到"连线"模式，先点一个接线柱(小圆点)，再点另一个，就连上导线。
  • 移动/改参数："选择"模式下拖动元件；点中元件后在右侧改它的阻值/电压；点开关可闭合或断开。
  • 删除：切到"删除"模式，点元件或导线即可删除。
  • 提示：电流表要串联、电压表要并联——接错了系统会立刻提示短路或读数异常，正好用来学。

🔬 伏安法测电阻 · 探究欧姆定律

初中电学最核心的实验。你要操作滑动变阻器改变电路、自己读电流表/电压表的指针、用控制变量法分别探究"I 与 U""I 与 R"的关系。

实验原理 & 器材

  • 原理：欧姆定律 I = U/R，测电阻 R = U/I。
  • 器材：电源、开关、滑动变阻器、待测/定值电阻、电流表、电压表、导线。
  • 电流表串联接入；电压表并联在被测电阻两端。
  • 滑动变阻器作用：①保护电路（先调到阻值最大再闭合开关）②改变被测电阻两端电压和电流，便于多次测量。

读数要点（中考高频失分点）

  • 先看量程（最大刻度），再看分度值（一小格代表多少）。
  • 电流表常见两量程：0~0.6A(分度0.02) 与 0~3A(分度0.1)。
  • 电压表常见两量程：0~3V(分度0.1) 与 0~15V(分度0.5)。
  • 指针超过量程会打表/读不准——要换大量程或增大变阻器电阻。

误差分析 & 注意事项

  • 多次测量取平均值是为了减小偶然误差（不是为了找规律时，求平均才有意义）。

💡 串联电路 与 并联电路

两盏灯怎么连，电流电压怎么分？拆掉一盏灯另一盏会不会灭？切换连接方式 + 断开开关，亲眼看现象。

核心概念

  • 串联：一条路。电流处处相等 I=I₁=I₂；电压分配 U=U₁+U₂；一处断，全灭。
  • 并联：各支路。电压相等 U=U₁=U₂；电流分流 I=I₁+I₂；一支断，其余照常。
  • 口诀：串分压、并分流。

🧲 通电螺线管 / 电磁铁

电生磁！电流越大、匝数越多磁性越强；改变电流方向，N、S 极对调。

核心概念

  • 电生磁：通电螺线管像条形磁铁。磁性强弱由电流×匝数决定（插铁芯增强）。
  • 安培定则：右手握螺线管，四指弯向电流方向，大拇指指 N 极。
  • 改变电流方向 → N/S 极对调。这是电磁铁"磁性有无、强弱、方向都可控"的原因。

🏠 家庭电路与安全用电

家里的电器都是并联在火线和零线之间。打开越多大功率电器，总电流越大——超过保险丝额定值就会熔断。还能演示短路、触电、漏电与接地保护。

家庭电路的组成

  • 进户线：火线 L（对地 220V）和 零线 N（对地 0V）。
  • 顺序：进户线 → 电能表（测耗电）→ 总开关 → 保险丝/空气开关 → 各用电器。
  • 用电器之间并联；开关要接在火线上、且与它控制的灯串联。
  • 保险丝接在火线上；三孔插座：左零、右火、上接地。

三大危险 & 保护

  • 过载：同时用太多大功率电器，总电流 I=ΣP/U 过大 → 保险丝熔断。
  • 短路：火线零线直接相连，电阻≈0，电流剧增 → 保险丝熔断。
  • 触电：人同时接触火线和大地，电流经人体入地（高于约 30mA 即危险）。
  • 接地保护：金属外壳接地线，漏电时电流走地线，人就安全（配合漏电保护器跳闸）。
    注意：保险丝只防过载/短路，不防触电！

⚡ 测量小灯泡的电功率

用电压表、电流表测出小灯泡两端电压 U 和电流 I，算出功率 P = UI。关键：灯丝是钨丝，温度越高电阻越大，所以不同电压下电阻、功率都不同。

实验原理 & 器材

  • 原理：P = UI；同时还能算 R = U/I。
  • 器材：电源、开关、滑动变阻器、电流表(串联)、电压表(并联在灯两端)、小灯泡。
  • 额定电压 U额：灯正常发光时的电压（本灯 2.5V）。此时的功率叫额定功率 P额。
  • 滑变作用：改变灯两端电压，观察从偏暗 → 正常 → 过亮，并测出额定功率。

为什么不能"求平均"（易错点）

  • 测定值电阻可以多次测量求平均；但小灯泡电阻随温度变化，不同电压对应不同的 R 和 P。
  • 所以小灯泡的电功率不能把几次的 P 求平均，要的是额定电压下那一组的功率。
  • 观察 R=U/I：电压越高、灯越亮、温度越高 → R 越大。

🔥 焦耳定律 · 电流的热效应

电流通过导体会产生热量：Q = I²Rt。用控制变量法探究 Q 跟电流 I、电阻 R、通电时间 t 的关系。

核心概念

  • 焦耳定律：Q = I²Rt（电流的平方 × 电阻 × 时间）。单位 Q 焦耳(J)。
  • 注意 I 是平方：电流加倍，发热变 4 倍——所以电流的影响最显著。
  • 探究方法：用串联让两个电阻通过相同电流，比较不同 R 的发热（控制 I、t 相同，看 Q 与 R）。
  • 纯电阻电路里 W=Q（电能全部变热）；电动机等非纯电阻 W>Q。

🛟 探究浮力大小 · 验证阿基米德原理

真实做法：用称重法测浮力（F浮=G−F示），再用溢水法测排开液体的重力，比较二者是否相等，从而验证 F浮 = G排。

实验原理

  • 称重法测浮力：F浮 = G物 − F示。
  • 排开液体的重力：G排 = ρ液·g·V排，用溢水杯+小桶称出。
  • 阿基米德原理：F浮 = G排 = ρ液·g·V排。
  • 结论关键：浮力大小只跟 ρ液 和 V排 有关，跟物体浸没深度（全浸后）、物重无关。

实验步骤

  1. 测物体在空气中的重力 G物（弹簧测力计）。
  2. 把物体浸入溢水杯，记测力计读数 F示，排出的水流入小桶。
  3. 算浮力 F浮=G物−F示；称小桶+水的总重，减去空桶重得 G排。
  4. 换不同深度/不同液体，多次实验，比较 F浮 与 G排。

⚖️ 探究杠杆平衡条件

调到杠杆水平平衡，记录多组 F₁、L₁、F₂、L₂，找出 F₁L₁ = F₂L₂ 的规律。

核心概念 & 为什么要"水平平衡"

  • 平衡条件：F₁·L₁ = F₂·L₂。
  • 实验前调杠杆在水平位置平衡，是为了让力臂正好等于支点到挂钩的格数，便于直接读力臂。
  • 分类：动力臂>阻力臂→省力(撬棍)；<→费力(镊子)；=→等臂(天平)。省力必费距离。

🔭 探究凸透镜成像规律 · 光具座

真实做法：在带刻度的光具座上固定透镜，移动蜡烛改变物距 u，再移动光屏直到像最清晰，读出像距 v，记录多组数据找规律。

实验原理 & 调节

  • 成像公式 1/u + 1/v = 1/f。
  • 实验前要把烛焰、透镜、光屏三者中心调到同一高度，否则像偏出光屏。
  • 实像能用光屏承接（倒立）；虚像不能承接，需透过透镜直接看（正立放大）。

成像规律表（要背会）

  • u>2f：倒立缩小实像（照相机）
  • u=2f：倒立等大实像（分界）
  • f<u<2f：倒立放大实像（投影仪）
  • u=f：不成像
  • u<f：正立放大虚像（放大镜）

中考真题演练

f=10cm，蜡烛距透镜 15cm，光屏上得到清晰像。该像是：
A. 倒立、放大的实像（投影仪）
B. 倒立、缩小的实像（照相机）
C. 正立、放大的虚像（放大镜）
D. 正立、等大的虚像
f=10→2f=20。u=15 满足 f<u<2f，成倒立放大实像（投影仪）。能用光屏承接→实像。答案 A。在光具座把 f=10、u=15，点自动对焦验证。
